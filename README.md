## API Reference

#### Get all character

```http
https://rickandmortyapi.com/api/character/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |

#### Get item

```http
 "https://rickandmortyapi.com/api/character/?page=20",
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
